# FruitCollector

A simple Roblox game made with Rojo.

## Setup
1. Install the Rojo plugin in Roblox Studio.
2. Run:
   ```bash
   rojo serve
   ```
3. In Roblox Studio, connect to the Rojo server.
4. Play test the game.

## How it works
- Fruits spawn randomly on the map.
- Players collect fruits by touching them.
- Players sell fruits in the sell zone to earn Coins.
- Messages appear in a small UI when collecting or selling.
